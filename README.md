# markdown
## markdown学习
- **什么是markdown**
  
  Markdown 是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）和亚伦·斯沃茨（Aaron Swartz）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML(或者HTML)文档”。
- **markdown的优点**
  
  纯文本，所以兼容性极强，可以用所有文本编辑器打开
让你专注于文字而不是排版
格式转换方便，Markdown 的文本你可以轻松转换为 html、电子书等
Markdown 的标记语法有极好的可读性
